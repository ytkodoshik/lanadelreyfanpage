#  Lana Del Rey Fan Page

Welcome to the **Lana Del Rey Fan Page** repository. This is a responsive, static website dedicated to the life, career, and music of Elizabeth Woolridge Grant, known professionally as Lana Del Rey.

##  About The Project

This project was created to showcase front-end development skills while paying tribute to the artist. It serves as a comprehensive hub for fans, featuring a biography, detailed discography, and multimedia content.

### Key Features
* **Hero Section:** Immersive landing page with a navigation menu and vintage aesthetic.
* **Biography & Career:** A timeline of the artist's journey from "Lizzy Grant" to global stardom.
* **Discography:** Detailed cards for major albums like *Born to Die*, *Ultraviolence*, and *Honeymoon*.
* **Multimedia Gallery:** A dedicated section for photos and music videos.
* **Live Clock:** A custom JavaScript feature in the footer displaying the current date and time in real-time.
* **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices (RWD).

##  Built With

* **HTML5** – Semantic structure.
* **CSS3** – Styling, Flexbox/Grid layouts, and responsive media queries.
* **JavaScript** – DOM manipulation for the real-time clock and interactive elements.
* **Font Awesome 4.7** – Social media and UI icons.
* **Google Fonts** – Typography using *Lobster* (headings) and *Roboto* (body text).

##  Run Locally

To view this project on your local machine:

1. Clone or Download the repository.
2. Open the folder on your computer.
3. Double-click the `index.html` file to launch it in your default web browser.

##  Design

The design is heavily inspired by Lana Del Rey's vintage Americana aesthetic, utilizing a color palette of deep red (`#a4040c`), white, and grey, paired with elegant typography.
